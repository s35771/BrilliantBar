# Pulsar Energy Bar

A modern 2026 landing page for Pulsar, a clean cognitive energy bar.

## Preview locally

This site is built with plain HTML, CSS, and JavaScript. No dependencies are required.

```bash
python3 -m http.server 4173
```

Then open <http://localhost:4173>.

## Files

- `index.html` - page structure and marketing copy
- `styles.css` - responsive visual system and product mockups
- `script.js` - mobile navigation, scroll reveal, and signup feedback
